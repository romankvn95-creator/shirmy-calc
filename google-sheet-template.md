# Google Таблица — Калькулятор ширм

## Структура (строка 1 = заголовки, начиная со строки 2 — данные)

| A | B | C | D | E | F | G | H | I | J | K |
|---|---|---|---|---|---|---|---|---|---|---|
| Имя менеджера | Токен | Активен | Телефон | Дата добавления | Ссылка | Комментарий | Последний вход | Посещений | Онлайн до | 🟢 Онлайн? |

**Столбец K** — формула для индикатора онлайн (скопируй в K2 и протяни вниз):
```
=ЕСЛИ(J2="";"";ЕСЛИ((ТДАТА()-J2)*1440<5;"🟢 онлайн";"⚫ не активен"))
```
*(Английская версия Sheets: `=IF(J2="","",IF((NOW()-J2)*1440<5,"🟢 онлайн","⚫ не активен"))`)*

🟢 показывает «онлайн» если человек был активен в последние 5 минут.

## Настройка доступа (обязательно!)

### 1. Опубликуй таблицу
Файл → Поделиться → Опубликовать в интернете → Выбери лист → Опубликовать

### 2. API-ключ Google (для чтения)
1. Перейди: https://console.cloud.google.com
2. Создай проект (или выбери существующий)
3. APIs & Services → Enable APIs → включи **Google Sheets API**
4. Credentials → Create Credentials → **API Key**
5. Нажми на ключ → API restrictions → выбери "Google Sheets API"
6. Скопируй ключ → это `VITE_SHEETS_KEY`

### 3. ID таблицы
Из URL таблицы: `https://docs.google.com/spreadsheets/d/**ВОТ_ЭТОТ_ID**/edit`
Это `VITE_SHEETS_ID`

## Управление менеджерами

| Действие | Как сделать |
|---|---|
| Добавить менеджера | Вставить строку: имя, токен, TRUE, телефон, дата, ссылка |
| Сгенерировать ссылку | `https://ДОМЕН/?t=ТОКЕН` (вставь в колонку F) |
| Отключить доступ | Колонка C → поменять TRUE на FALSE |
| Включить обратно | Колонка C → поменять FALSE на TRUE |
| Видеть онлайн-статус | Смотреть колонку K (обновляется каждые 2 мин) |
